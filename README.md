# 🚀 ResumeFlow - Homepage & Login UI

A modern and responsive **ResumeFlow** landing page built using **HTML5, CSS3, and JavaScript**. This project includes
an attractive homepage inspired by modern SaaS websites and a fully styled login page designed with a dark theme.

## 📌 Features

- 🏠 Modern Homepage
- 🔐 Responsive Login Page
- 🎨 Dark Theme UI
- 📱 Mobile-Friendly Design
- ✨ Smooth Hover Effects & Animations
- 🖼️ Hero Section with Image
- 📊 Feature Highlights
- ⚡ Lightweight JavaScript Interactions
- 🧩 Semantic HTML5 Structure
- 🎯 Clean and Organized CSS
